## Requirement

1. go >=1.20
2. ngork or devtunnel
3. github app
4. .env file to store your OpenAI config
   - `OPENAI_API_BASE`
   - `OPENAI_API_TYPE`
   - `OPENAI_API_VERSION`
   - `OPENAI_MODEL_NAME`
   - `OPENAI_API_KEY`

## How to Use

1. go get
2. go build
3. ./pull-request
4. ngrok http 8085
